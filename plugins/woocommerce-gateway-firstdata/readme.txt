== Clover for WooCommerce
Author: kestrelwp
Tags: woocommerce
Requires PHP: 7.0
Requires at least: 5.2
Tested up to: 6.5.3

Accept credit cards in WooCommerce through Clover, First Data Payeezy (formerly GGe4), or Global Gateway

See https://woocommerce.com/document/clover/ for full documentation.

== Installation ==

1. Upload the entire 'woocommerce-gateway-firstdata' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
